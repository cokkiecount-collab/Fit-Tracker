# Features

## Dashboard

The dashboard should display:

- Recent workouts
- Recent PRs
- Total tonnage
- Workouts this month

---

## Workout Logging

### Main Goal

Logging a workout should be fast and simple.

Example:

Bench Press

Last workout:
75 kg × 8 reps

Set 1
75 kg × 8

Set 2
75 kg × 8

Set 3
72.5 kg × 10

---

## Exercise History

Users should be able to see:

- Previous workouts
- Progression over time
- Personal records

---

## Personal Records

Track:

- Highest weight
- Best volume
- Best estimated 1RM

---

## Statistics

Track:

- Total tonnage
- Workouts completed
- Most trained exercises
- Weekly volume
- Monthly volume
